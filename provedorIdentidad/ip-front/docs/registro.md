# Registro
 
Para implementar el Servicio de Identidad Digital en un sistema es necesario primero realizar un registro de manera personal con el área encargada.
 
Los datos que se necesitan son los siguientes:
 
- Nombre del aplicativo o sistema
- Url de respuesta
- Dominio del aplicativo
 
 Ejemplo:

 * Nombre del aplicativo o sistema: OMEXT
 * Url de respuesta: (http://omext.funcionpublica.gob.mx/system-log-in-ip)
 * Dominio del aplicativo: omext.funcionpublica.gob.mx

Con estos datos se generará un **Secret Key** y un **ID del aplicativo**, los cuales son necesarios para la implementación del servicio.
